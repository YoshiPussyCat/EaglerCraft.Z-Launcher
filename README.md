# EAGLAUNCHER

A launcher created to improve your gameplay experience with EaglerCraft, an open-source Minecraft that runs in browsers.

---

## 📌 Project Overview

EAGLAUNCHER is a web-based launcher that allows users to access different EaglerCraft versions through a clean and interactive interface, without requiring downloads or installations.

It includes multiple official versions as well as modded clients. This project was heavily inspired by irv77.

## 📂 Project Structure


/ (root)

├── index.html

├── SFX/

├── images/

└── Fonts/


If the website does not work properly, the launcher can be downloaded as a `.zip` file.

To use it locally:
1. Extract the ZIP file  
2. Ensure the folder structure matches the one above  
3. Open `index.html` in your browser
